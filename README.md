# Porturl Android Client

This app is an OIDC (OpenID Connect) client for the Porturl API, built with Android and Kotlin.

## Features
- OIDC login (AuthManager)
- Fetches applications from Porturl API (ApplicationApi)
- Secure token storage
- Basic sample UI
- Real unit and UI tests

## Getting Started

1. Clone the repo
2. Add your configuration to `local.properties` or environment variables
3. Build and run with Android Studio or `./gradlew assembleDebug`
4. Run tests with `./gradlew test` for unit tests, `./gradlew connectedAndroidTest` for UI tests
